## 🎯 About The Project

**Tajmer** (Polish for *Timer*) is a minimalist, distraction-free timer built for the command line. Unlike heavy GUI applications, Tajmer runs lightweight directly inside your favorite terminal screen, keeping system resources low and your focus high.

### Why use Tajmer?
* 🚀 **Zero Dependencies:** No bloat. Pure, native Python code.
* 📦 **Portable:** Just drop the script anywhere and run it.
* 💻 **Dev-Friendly:** Perfect for terminal-centric workflows and Pomodoro sessions.

---

## ⚙️ Requirements

Tajmer keeps things incredibly simple. You only need a standard Python installation:

* **Python 3.6** or greater
* No external libraries required (`pip install` is not needed!)
